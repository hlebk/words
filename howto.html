How to use

Preparation

To start using this app, you have to prepare a valid text file containing words and their translations. Here is how it should look like:
to dream : śnić, marzyć
human : człowiek
flower, blossom : kwiat

  1. The file can contain as many lines as needed.
  2. Each line consists of one or more words with the same meaning from both languages.
  3. Languages are separated by a colon.
  4. Synonyms are separated by a comma.
  5. Empty or incomplete lines are not allowed.
  6. File should be saved with .txt extension.

Usage

  1. Import your file.
  2. Choose one:
    • Practice - you will be asked if you remember a random word from your file. If you answer "yes", the word won't appear again.
    • Test - you will be prompted to write only one correct translation of a random word from your file. The result window will appear after all questions have been answered.
  3. Check the checkbox below if you want to reverse the order of languages in questions.
  4. Click the Start button and enjoy.
